# Интернет-магазин на Flask

## Описание

Это учебный проект интернет-магазина, реализованный на Python с использованием Flask, SQLAlchemy и Jinja2. Приложение поддерживает каталог товаров, оформление заказов, гостевую книгу, мониторинг CPU, калькулятор доставки, электронную таблицу и административную панель для управления товарами.

---

## Основные возможности

- Каталог товаров с фильтрацией и сортировкой
- Просмотр детальной информации о товаре
- Оформление заказа с выбором адреса на карте
- Гостевая книга для отзывов
- Калькулятор стоимости доставки по городам
- Мониторинг загрузки CPU (с использованием внешнего API)
- Электронная таблица (локально в браузере)
- Админ-панель (добавление, редактирование, удаление товаров)
- Загрузка изображений товаров

## Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone <адрес-репозитория>
   cd "Сайт магазина"
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   venv\Scripts\activate  # Windows
   # или
   source venv/bin/activate  # Linux/Mac
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите приложение:**
   ```bash
   python app.py
   ```
   По умолчанию сайт будет доступен на http://127.0.0.1:5000/

## Важно
- Для работы некоторых функций требуется интернет (API доставки, CPU, города).
- Все изображения товаров хранятся в `static/images/products/`.
- Кэш городов для доставки — файл `city_cache.json`.
- Счетчик посещений random-color — файл `random_color_visits.txt`.
- Базы данных: `shop.db` (товары), `guestbook.db` (гостевая книга).
- Для корректной работы загрузки изображений убедитесь, что папка `static/images/products/` существует и доступна для записи.

## Советы по эксплуатации
- Не храните реальные пароли и секретные ключи в коде — используйте переменные окружения.
- Для деплоя на сервер используйте gunicorn/uwsgi и настройте nginx.
- Для продакшена отключите режим `debug=True`.
- Регулярно делайте резервные копии баз данных.
